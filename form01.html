Начало работы с формами.

В этом примере мы рассмотрим самую простую форму. И убедимся, что данные введенные в форму могут исчезнуть при различных событиях.

    Источники:
  1. Файл примера с сервера GitHub
  2. Файл конспекта: 01 Форма потеря данных данных.doc
    Что смотрим:
  1. Форма - это контейнер для других тегов
  2. Предназначение формы: позволить пользователю вводить информацию на веб-странице.
  3. Убеждаемся, что данные введенные в форму могут исчезнуть даже после стандартных событий в браузере:
    • Обновление страницы
    • Закрытие текущей вкладки браузера.
    • Закрытие браузера.
    • Перезагрузка компьютер.
Для этого вводим любые данные в форму и смотрим хотя бы несколько из вышеуказанных событий.
